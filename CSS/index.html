    CSS

    1、盒模型

    css的盒模型有两种标准盒模型和IE盒模型

    • 标准盒模型: 内容的宽高为content的宽高
    • IE盒模型: 内容的宽高为content + padding + border的宽高
    更改盒模型
    盒模型margin负值问题

    margin为负值的时候会缩小盒子宽高，影响到其他的元素

    2、块级格式化上下文(BFC)

    块级格式化上下文(BFC)是一个独立渲染的区域(一个隔离的独立容器)，让处于BFC内部的元素与外部元素互相隔离

    BFC的特性
    • 同一个BFC的相邻的两个元素会垂直排列
    • 同一个BFC的相邻的两个元素margin会重合
    • BFC的区域不会与float元素重叠(float也是一个BFC)
    • BFC在页面中是一个独立的容器，内部的元素不会影响到外部的元素
    • 计算BFC高度的时候，内部的float浮动元素也参与计算
    触发BFC的方式
    • position: absolute/fixed绝对定位
    • display: inline-block/table/flex/inline-flex等
    • float: 除了none以外的值
    • overflow: 除了visible以外的值
    BFC的运用
    • 阻止margin重叠
    • 清除浮动
    • 自适应两栏布局: 左float + 右BFC,利用浮动与BFC不会重叠的机制
